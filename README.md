# DSA Implementations

This repository contains implementations of various Data Structures and Algorithms across multiple programming languages.

## Current Implementations

### Linked Lists
A **Circular Doubly Linked List** is implemented in the following languages:

*   **Python** (`linkedLists/main.py`): Fully documented implementation using pythonic conventions (`__len__`, `__str__`).
*   **JavaScript** (`linkedLists/main.js`): Class-based implementation.
*   **TypeScript** (`linkedLists/main.ts`): Generic implementation (`LinkedList<T>`).
*   **C++** (`linkedLists/main.cpp`): Implementation using raw pointers.

**Features:**
*   `push`/`push_back`: Add to end.
*   `insert`: Add at specific index.
*   `delete`/`delete_by_index`: Remove at specific index.
*   `get_node`: Access node at index.
*   `contains`/`includes`: Check for value existence.
